E-Commerce Application

A full-stack e-commerce application built to learn how modern applications work internally from end to end.

Project Goal

The primary objective of this project is to understand the complete software development lifecycle by building a real-world application from scratch.

The application will be developed in phases:

1. System Design
2. Database Design
3. API Design
4. Backend Development
5. API Testing (Swagger & Postman)
6. Frontend Development
7. API Integration
8. Deployment

Rather than focusing only on UI development, this project aims to understand how frontend, backend, database, authentication, and deployment work together.
